# Portfolio Website

A static portfolio website built for GitHub Pages.

## Deploy

Publish from the repository root using GitHub Pages with the branch set to `main` and folder set to `/root`.

## Custom domain

After buying a domain, add a `CNAME` file containing the domain name, then set DNS records at the registrar:

- `A` records for `@`: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
- `CNAME` for `www`: your GitHub Pages hostname
